Version 1.0

Featured

Since we have now brought back TimerBriber, we have made the following adjustments:

  • The patch is back to 1.0.
  • Font adjustments:
    • Everything is now smaller than usual. Quality over quantity.
    • Hovering over a list now changes the background of the line of text. Hover over me!
  • Navigation bar adjustments:
    • Changed background colour from blue to a darkish grey.
    • Hovering over text will change the background & text colour. Neat!
  • Team adjustments:
    • Images are now next to the name of the team member. (they choose which image that's appropiate)
    • Tweaks to animations on names.
    • Clicking on a team member's name now takes you to their GitHub profile.
  • Home adjustments:
    • Header is now in blue. Might be reverted in a future patch.
    • Updated the following images both in the Home and other pages:
      • Rocket League
      • Black Ops 4
      • TimerBriber
  • Fixed an issue where people from other time zones would get delayed countdowns for something like Fortnite's battle pass.
